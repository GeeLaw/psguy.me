Naming

It means, literally, Windows console.

Notice

You should NOT use this script unless you use Chinese in Windows console. You have to acquire YaHei Consolas Hybrid font yourself.

Function

The script:

  1. Notices you to install the specified font;
  2. Removes all current console settings and sets the default as that to Gee's preference.

Configurability

The script is NOT configurable.
